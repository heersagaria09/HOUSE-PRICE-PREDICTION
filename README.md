This project predicts house prices using Machine Learning based on features like area, number of rooms, and other factors.

## Features

* Data preprocessing and cleaning
* Feature analysis
* Price prediction using regression

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* Matplotlib

## Model Used

Linear Regression

## Description

This project demonstrates how machine learning can be used to predict real estate prices based on input features.
